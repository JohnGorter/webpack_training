# Setup

---
### Getting started

What you need:
- NodeJS


---
### Install WebPack
To install run
```
mkdir webpack-demo
cd webpack-demo
npm init -y
npm install webpack --save-dev
npm install webpack-cli --save-dev
```



---
### Test Webpack

We can test the installation by typing in the Terminal:
```
./node_modules/.bin/webpack --version
```
